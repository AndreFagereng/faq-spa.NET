Informasjon:
I denne oppgaven har jeg brukt Angular med WebApi i ASP.NET.
En kunde kan kun sende inn spørsmål til CustomerFAQ, og ikke vanlig FAQ.
Kunden kan også Upvotes / Downvote ulike CustomerQuestions, og kan hente ut
top 5 spørsmål basert på Upvotes.

For å kjøre løsningen:
Unzip og kjør løsningen(Dette er testet, skal fungere)
FAQ spørsmål er seedet fra før
CustomerFAQ spørsmål er ikke seedet og må laget manuelt.

Angular 5
ASP.NET CORE 2.1
RESTFUL WebApi
